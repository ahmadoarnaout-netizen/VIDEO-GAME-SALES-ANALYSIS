# 🎮 Video Game Sales Analysis in R

## 📌 Project Overview
This project analyzes global video game sales data using R programming language and data visualization techniques to discover trends, top-performing genres, publishers, platforms, and regional sales patterns.

The analysis includes data cleaning, exploratory data analysis (EDA), and visualization of important insights from the dataset.

---

## 📊 Objectives
- Analyze global video game sales
- Identify top-selling games and publishers
- Compare sales across different regions
- Discover trends in genres and gaming platforms
- Create visualizations for better insights

---

## 🛠️ Technologies Used
- R
- tidyverse
- ggplot2
- dplyr
- readr

---

## 📂 Dataset
The dataset contains information about:
- Game Name
- Platform
- Genre
- Publisher
- Release Year
- Regional Sales
- Global Sales

---

## 📈 Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sales Trend Analysis
- Genre Analysis
- Platform Performance Comparison
- Regional Sales Visualization

## 📷 Visualizations
The project includes charts and graphs such as:
- Top Selling Games
- Sales by Genre
- Platform Performance

- Regional Sales Distribution

1. Clone the repository:
```bash
git clone https://github.com/yourusername/video-game-sales-analysis.git
```

3. Install required packages:

4. Run the script.
